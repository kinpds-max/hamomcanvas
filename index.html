하맘캔버스 단축키 & 도움말

탐색 (Zoom & Pan)

  • Space + 마우스 드래그 : 화면이 어디서든 자유롭게 이동합니다.
  • 마우스 휠 : 화면 확대 / 축소 (마우스 포인터 중심)
  • Ctrl + 마우스 휠 : 화면 확대 / 축소 (강제 또는 브라우저 줌 방지 상태)
  • 마우스 휠 버튼 클릭 + 드래그 : Space 없이 즉시 화면 이동
  • Shift + 1 : 모든 카드가 보이게 화면 크기 맞춤 (Zoom to Fit)

카드 생성 및 관리

  • 캔버스 더블클릭 : 더블클릭한 빈 자리에 즉시 새 텍스트 카드 생성
  • V : 선택/이동 도구로 전환
  • H : 화면 이동(Pan) 도구로 전환
  • C : 연결 도구로 전환 (한 카드를 클릭한 뒤 다른 카드를 클릭해 연결)
  • Delete 또는 Backspace : 선택된 카드 삭제

로컬 백업 및 보안

  • 모든 데이터는 브라우저의 localStorage에 자동 저장되어 새로고침해도 유지됩니다.
  • 비밀번호는 안전하게 해시(암호화)되어 저장됩니다.
  • 브라우저 초기화에 대비해 '내보내기' 버튼으로 데이터를 JSON 파일로 주기적으로 저장할 수 있습니다.
